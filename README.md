🚗 Parquímetro

Projeto desenvolvido em HTML, CSS e JavaScript utilizando Programação Orientada a Objetos (POO).

📖 Descrição

Esta aplicação simula um parquímetro. O usuário informa um valor em reais e o sistema calcula automaticamente o tempo de permanência no estacionamento e o troco, quando houver.

⚙️ Funcionalidades
Recebe um valor informado pelo usuário.
Calcula o tempo de permanência:
R$ 1,00 → 30 minutos
R$ 1,75 → 60 minutos
R$ 3,00 → 120 minutos
Calcula e exibe o troco, se houver.
Exibe a mensagem "Valor insuficiente" quando o valor é menor que R$ 1,00.
🛠️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript (POO)
